# BlockSweeper

Monorepo ringan untuk MiniPay Mini App di Celo.

## Struktur

- `apps/web`: frontend React + Vite untuk MiniPay
- `apps/api`: placeholder backend untuk session, validation, leaderboard
- `packages/shared`: placeholder type dan util bersama
- `contracts`: placeholder smart contracts dan deployment scripts

## Workspace

Project ini memakai `pnpm` workspace agar setup tetap sederhana selama hackathon, tapi tetap memberi pemisahan jelas antara web, backend, shared, dan contracts.
